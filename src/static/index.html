System-Status & Auswertung

Das DSGVO Privacy Gateway läuft lokal und schützt Ihre LLM-Kommunikation. Jede PII-Erkennung läuft vollständig auf Ihrem System, bevor Daten ins Internet gesendet werden.

Integration & API Nutzung

Das DSGVO Privacy Gateway ist als Drop-In Proxy konzipiert. Du kannst deine bestehenden Anwendungen (z.B. LangChain, OpenAI SDK, LlamaIndex) einfach auf dieses Gateway umleiten.

1. OpenAI SDK (Python)

Ändere einfach den base_url Parameter im OpenAI Client. Der Proxy maskiert automatisch den Prompt und de-anonymisiert die Antwort im Rückkanal.

from openai import OpenAI

# Verbinde zum lokalen DSGVO-Proxy statt direkt zu OpenAI
client = OpenAI(
    base_url="http://localhost:8000/v1",
    api_key="Dein-OpenAI-API-Key"  # Wird an OpenAI weitergereicht
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "user", "content": "Mein Name ist Max und ich wohne in München."}
    ]
)

print(response.choices[0].message.content)
# Ausgabe: "Hallo Max! Wie kann ich dir in München helfen?"
# Hinweis: Das OpenAI-Modell sah nur: "Mein Name ist  und ich wohne in ."

2. cURL Request

Das Gateway verhält sich exakt wie die OpenAI API v1.

curl http://localhost:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {"role": "user", "content": "Rückmeldung für Herrn Albert Einstein (albert@einstein.org)."}
    ]
  }'
